Table Tags

Tag Name Description
<table> Table The wrapper element for all HTML tables.
<thead> Table Head The set of rows defining the column headers in a table.
<tbody> Table Body The set of rows containing actual table data.
<tr> Table Row The table row container.
<td> Table Data A cell containing table data.
<tfoot> Table Foot The set of rows defining the footer in a table.

Table Attributes

Attributes Name Description
colspan Column Span Defines how many columns a td element should span.
rowspan Row Span Defines how many rows a td element should span.


Commonly used HTML tags

Tag Description
<!DOCTYPE> first line of code on any html page, defines the file type.
<html> Hyper Text Markup Language.
<head> The head or prologue, of the HTML Documnet
<title> Assigns website or webpage name to the browser tab.
<!--> This is the code you type in order to make or leave a comment.
<href> Hyper Text Reference, used to link things such as img src or our css stylesheet into your website
<ul> Unordered list, used to create a list with bullets vs numbers
<ol> Ordered list, used to create a list with numbers vs bullets.
